Part 4:
RAM Installation

This next step is installing the computer memory

Inserting RAM into proper slots is key

RAM is a sensitive component and as such should be handled with care.
It's recommended that contact is avoided with the contact pins on the underside of each module.

  • If you have two sticks of RAM, you want to take advantage of the dual-channel memory.

  • These sticks should be installed in channels on the motherboard with matching colors, usually labeled A1 and B1, though sometimes A2 and B2 are preferable.

  • Check your motherboardâ€™s manual to confirm which are best for your system.

  • Once you know where to insert the RAM, you want to unclip the RAM locks on the motherboard module slots.

  • Then match the RAM module notches to those found in the slots on the motherboard.

  • Next up, insert the RAM carefully, using light even force across the stick.

  • After it inserts into the slot, push down on either side of the RAM module until it securely clicks into place.
