Part 5:
Motherboard Installation

With the CPU, the cooler, and RAM modules installed, the motherboard is ready to be installed!

First, we need to install the I/O shield, a long piece of metal that has cutouts for the all the rear ports and connections.

Depending on if you have a ATX or mini ATX build, your motherboard installation and fit might be a bit different.

I/O Shield

Motherboard

  • At the back, or the top of your PC case, you’ll see a rectangular cut-out. This is for the motherboard’s I/O panel — the portion containing the USB, video, and Ethernet connections.

  • Your motherboard will be packaged with an I/O shield that fits into this rectangular cut-out.

  • Install it from the inside of the case facing outwards. It should snap onto place fairly easily.

  • If you install that shield, and then align your motherboard’s I/O panel with it, you’ll find the screw holes in the motherboard align with the stand-offs in your case.

  • Tighten the screws, but do not overtighten them — proceed in a star pattern, tighten each screw a little bit at a time, just enough to secure the board in place.

  • Check the sides of the motherboard by lightly pulling to make sure every inch has been secured. It shouldnt be wiggling at this point.
