Finishing up!

The final steps to completing your build

You now want check if you did everything properly and install the operating system

You will want a USB or CD with the OS on it to properly install it.

Did it work?

Operating System

Congratulations!

  • Switch on the PSU switch and turn on your computer

  • If everything worked properly you'll be in the BIOS POST screen, asking for an OS to be installed if one is not detected.

  • You'll now need to follow OS installation instructions.

  • In the BIOS menu tell your PC to boot from either a USB drive or CD with windows installed on it.

  • From there you can go through the instructions to install the operating system onto your new PC.

  • All there is left to do is update your drivers, the OS (Windows 10)will automatically download and install the rest of your drivers in most cases. Check the “Update & Security” menu in the Settings pane for more information regarding this process.

  • If that doesnt work, then going online and downloading your required drivers for your motherboard and GPU is required.

    • If everything went smoothly, then Congratulations!!


    You have built your first PC!
